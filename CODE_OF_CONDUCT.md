# Code of Conduct

This project follows the Contributor Covenant, version 2.1.

The full text is published at https://www.contributor-covenant.org/version/2/1/code_of_conduct/.

## Scope

The Covenant applies in all project spaces, and also when an individual is
representing the project in public spaces.

## Reporting

To report a concern, open a private advisory at
https://github.com/Rsan0948/digital-sangha/security/advisories or contact the
maintainers directly. Reports are reviewed promptly and treated confidentially.

## Enforcement

Maintainers are responsible for clarifying and enforcing the standards
described in the Covenant. Outcomes follow the four-stage enforcement ladder
documented in section 2.1 of the Covenant text linked above.
